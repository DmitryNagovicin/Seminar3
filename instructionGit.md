# **Инструкция по работе с системой контроля версий Git**

## Что такое Git?

Гит это система контроля версий. Нужна для отслеживания и ведения истории изменения файлов, в вашем проекте.

## Узнаем о следующих командах:

- git status
- git add
- git commit
- git commit -a
- git commit -am
- git log
- git --oneline --all
- git diff
- git checkout
